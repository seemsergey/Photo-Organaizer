# Media Sorter (Сортировщик Медиа) 📸🎥

Утилита на Python с графическим интерфейсом (Tkinter) для автоматической организации ваших фото и видео архивов. Программа читает метаданные файлов (EXIF для фото и заголовки для видео) и аккуратно раскладывает их по папкам: `Год / Месяц`.

## ✨ Особенности

- **Поддержка Фото:** `.jpg, .jpeg, .png, .tiff, .heic, .webp` и др.
- **Поддержка Видео:** `.mp4, .mov, .avi, .mkv, .wmv` и др.
- **Умная сортировка:** Использует реальную дату съемки из метаданных, а не дату изменения файла в Windows.
- **Два режима работы:**
  - **Копирование:** Файлы остаются на месте, создаются их копии в новых папках.
  - **Перенос (Вырезать):** Файлы перемещаются в новые папки (освобождает место).
- **Предварительный анализ:** Программа покажет, сколько файлов имеют метаданные, а сколько — нет, до начала процесса.
- **Безопасность:** Если файлы имеют одинаковые имена, программа автоматически добавит временной штамп, чтобы избежать перезаписи.
- **Авто-установка зависимостей:** Программа сама проверит и установит нужные библиотеки при первом запуске.

## 🚀 Как запустить

1. **Убедитесь, что у вас установлен Python 3.7 или выше.**
2. **Склонируйте репозиторий:**
   ```bash
   git clone https://github.com/ВАШ_ЛОГИН/Photo-Organaizer.git
   cd Photo-Organaizer
   ```
3. **Запустите приложение:**
   ```bash
   python main.py
   ```

## 🛠 Технологии

- **Python 3**
- **Tkinter** — графический интерфейс.
- **Pillow** — работа с EXIF данными фотографий.
- **Hachoir** — извлечение метаданных из видеофайлов.
- **Shutil/OS** — работа с файловой системой.

## 📖 Как пользоваться

1. **Запустите программу.**
2. **Выберите типы файлов:** Отметьте галочками "Фотографии" и/или "Видеофайлы".
3. **Анализ (опционально):** Нажмите кнопку "Анализ", выберите папку, и программа скажет, сколько медиафайлов она нашла.
4. **Сортировка:**
   - Нажмите "Сортировка".
   - Выберите режим: "Копировать" или "Перенести".
   - Выберите **исходную папку** (где сейчас лежат файлы).
   - Выберите **целевую папку** (куда программа создаст структуру папок).
5. **Результат:** Программа создаст структуру вида:
   ```text
   Целевая папка/
   ├── 2023/
   │   ├── Январь/
   │   │   └── photo_1.jpg
   │   └── Февраль/
   │       └── video_1.mp4
   └── Неизвестная_дата/
       └── old_photo.jpg
   ```

## ⚠️ Примечание

Файлы без метаданных (например, сохраненные из WhatsApp или Telegram, которые стирают дату съемки) будут перемещены в папку **"Неизвестная_дата"**, чтобы вы могли разобрать их вручную.
